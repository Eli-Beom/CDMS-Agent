Browser Inspect / Row-State Harness

Date: 2026-05-27 (_260527) · Work Unit: daemon/extension inspect

목적은 현재 브라우저 화면에서 row가 실제로 입력 가능한지 판단해, 조건부 비활성 필드에서 자동화가 멈추지 않게 하는 것입니다.

개발 의도

  • CRF 코드만으로는 현재 subject/visit/page의 UI 활성 상태를 알 수 없습니다.
  • 가임 여부 = 예 후 비가임 사유가 비활성화되는 식의 상태 변화를 browser inspect로 확인합니다.
  • row label과 option을 분리해 “어떤 row를 누를지”와 “무엇을 선택할지”를 구분합니다.

추가된 Snapshot 필드

structuredRows: [
  {
    rowLabel: "성별",
    visible: true,
    editable: true,
    disabled: false,
    controlType: "radio",
    options: ["남성", "여성"]
  }
]
